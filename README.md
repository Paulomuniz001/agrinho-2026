Agrinho 2026:EcoManejo Hídrico: O Jogo do Equilíbrio Sustentável 

1 - Descrição do projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação, subcategoria Ensino Médio. O site aborda o tema “Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”, apresentando reflexões e informações sobre a gestão responsável dos recursos hídricos na agricultura através de uma simulação gamificada interativa, que demonstra o impacto direto do manejo de irrigação na saúde das plantas e na preservação do solo. .

2 - Objetivo

O objetivo deste projeto é conscientizar o usuário sobre a importância da agricultura de precisão. Por meio de uma mecânica de jogo, o usuário aprende que o desperdício (encharcamento) e a escassez (seca) são prejudiciais, sendo necessário encontrar o ponto de equilíbrio para alcançar uma colheita bem-sucedida e sustentável. 

Público-alvo

O projeto é destinado a estudantes, professores e comunidade em geral interessados em compreender a importância do equilíbrio entre a produção agropecuária e a preservação ambiental. 

3 - Estrutura do projeto

O projeto é composto pelos seguintes arquivos e pastas:

index.html: arquivo principal com a estrutura da página que renderiza o canvas da aplicação;
style.css: arquivo responsável pela aparência, centralização e organização visual do site;
script.js: arquivo responsável por toda a lógica da simulação, física da água, gerenciamento dos estados do jogo e renderização gráfica interativa (utilizando a biblioteca p5.js);
README.md: documentação do projeto;
assets: pasta utilizada para armazenar a biblioteca p5.js localmente.

5 - Funcionalidades

O site apresenta as seguintes funcionalidades:

Organização de conteúdos sobre o tema Agrinho 2026;
Tela de Introdução: menu inicial explicativo que permite ao usuário entender as regras de manejo hídrico antes de começar;
Contagem Regressiva: sistema dinâmico de 3 segundos para preparação do jogador;
Simulador de Ecossistema: 3 canteiros independentes que simulam plantas reais (Girassol, Flor do Campo e Flor Azul) com taxas de evaporação e necessidades hídricas diferentes;
Visual Adaptativo: o solo escurece ou clareia em tempo real conforme a umidade, e as plantas reagem visualmente (murcham na seca, apodrecem e caem se alagadas, e florescem com pétalas exclusivas ao atingirem o tempo de colheita);
Botão Tente Novamente / Reiniciar: elemento interativo desenvolvido em JavaScript que redefine as variáveis do ecossistema, permitindo que o usuário aprenda com os erros de manejo anteriores;
Layout dinâmico organizado com CSS.
